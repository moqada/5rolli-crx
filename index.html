    5rolli-crx

    Chrome Extension for 5ROLLI

    Preview

    Installation

    1. Downloads latest release file from GitHub Releases
    2. Open Google Chrome
    3. Drop 5rolli.crx to chrome://extensions.

    Build

    $ git clone https://github.com/moqada/5rolli-crx.git
$ cd 5rolli-crx
$ npm i && npm run build
$ cd build
